## Descripcion
Acciones seguras para Outlook Classic: cerrar proceso, modo seguro, resetnavpane, cleanviews y panel de perfiles.

## Que hace este script
Genera comandos BAT y PS1 basados en la plantilla "V5 Pro - Reparar Outlook basico" con las opciones configuradas por el usuario en ScriptForge 404.

## Motivos del nivel de riesgo
- Algunas opciones resetean vistas o paneles de Outlook del usuario.

## Requisitos previos
- Cerrar correos en edición.
- Confirmar que es Outlook Classic.

## Posibilidad de rollback
Las vistas/paneles pueden requerir configuración manual de nuevo.
