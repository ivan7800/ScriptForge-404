# ScriptForge 404 v5.4 — Advanced CAU Script Pack

Este paquete contiene 60 scripts standalone nuevos: 60 BAT, 60 PS1 y 60 README/checklist.

Los scripts standalone se generan en modo diagnóstico/seguro por defecto cuando la plantilla incluye reparación. Para ejecutar cambios reales, genera el script desde la app marcando "Ejecutar reparación/cambios reales" o revisa manualmente el bloque correspondiente.

Categorías incluidas: red avanzada, unidades de red, Office/Outlook, Teams/WebView2, Intune/Autopilot, SCCM, impresoras y tickets/evidencias.
