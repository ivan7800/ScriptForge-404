## Descripcion
Limpia cache de New Teams y Teams clasico para el usuario actual, con cierre previo de procesos.

## Que hace este script
Genera comandos BAT y PS1 basados en la plantilla "V5 Pro - Limpiar cache Teams" con las opciones configuradas por el usuario en ScriptForge 404.

## Motivos del nivel de riesgo
- Cierra Teams y borra caches locales regenerables.

## Requisitos previos
- Cerrar llamadas/reuniones.
- Confirmar que el usuario puede volver a iniciar sesion si se le solicita.

## Posibilidad de rollback
La cache se regenera al abrir Teams.
