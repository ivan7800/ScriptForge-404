# ScriptForge 404 v5.2.2 — Scripts standalone V5 Pro

Este paquete contiene 30 scripts BAT, 30 scripts PS1 y 30 README/checklist generados desde las plantillas V5 Pro.

## Nota importante

Los scripts standalone se entregan en modo seguro/diagnostico por defecto cuando la plantilla incluye acciones de reparacion o cambios reales. Esto evita modificaciones accidentales en equipos de usuario.

Para generar una variante de reparacion real:

1. Abre `index.html`.
2. Selecciona la plantilla V5 Pro.
3. Revisa el informe de riesgo.
4. Activa el campo de reparacion/cambios reales solo si procede.
5. Exporta un nuevo BAT/PS1 y pruebalo primero en VM o equipo piloto.

- `PS1/`: 30 scripts .ps1.
- `README/`: documentacion por script.
- `manifest.json`: indice tecnico de scripts.
